Installation Guide

Web browsers cannot talk directly to USB hardware. To print physical receipts, run the lightweight companion agent on your local machine.

1

Download the Agent Script

Download the Python script and requirements file to the cashier's computer.

# Save to a new folder on your PC pos_agent.py requirements.txt
2

Install Dependencies

Open a terminal or command prompt in that folder and install the required packages.

pip install -r requirements.txt
3

Configure and Run

Open pos_agent.py in a text editor. Change PRINTER_TYPE to match your setup ('usb' or 'network'). Enter your printer's IP or USB Vendor ID. Then run it:

python pos_agent.py

The script must be kept running in the background while the cashier is working. You can minimize the terminal window.
